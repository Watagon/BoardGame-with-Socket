# 課題

X11と非同期なソケット通信を用いたイベント駆動型プログラミングによる4目並べの実装

# 設計と実装

プログラムの全体的な構造として

- 4目並べに必要な関数
- 表示及び通信に必要な関数

の2つに分けた。

画面をゲームの途中で閉じてもそれをハンドルしてプログラムを正常終了させるようにする。

# コンパイル方法

```bash
gcc connect4_front.c connect4.c -lX11
```

また、別の方法として `cmake` を用いたコンパイルの自動化も行った

```bash
mkdir build
cmake ..
make
```

# 評価
